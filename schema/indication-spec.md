# 适应症索引格式规范 (indication-spec.md)

本文件定义 `indication_dir/` 目录下适应症索引文件的格式规范。

---

## 文件命名规则

- **格式**: `{适应症名称}.md`
- **示例**: `NSCLC_1L.md`, `胃癌一线.md`, `HER2阳性乳腺癌.md`
- **特殊字符处理**:
  - 空格 → 下划线
  - 括号 → 移除
  - 斜杠 → 下划线
- **治疗线数区分**:
  - 一线：`{适应症}_1L` 或 `{适应症}一线`
  - 二线：`{适应症}_2L` 或 `{适应症}二线`
  - 三线+：`{适应症}_3L+` 或 `{适应症}后线`
  - **不同线数视为不同适应症，分开文件**

---

## Frontmatter 规范

```yaml
---
indication: {适应症名称}      # 必需，与文件名一致
category: {适应症类别}        # 可选，如"NSCLC"、"胃癌"
line: {治疗线数}             # 可选，如"1L"、"2L"、"3L+"
aliases: [{别名1}, {别名2}]  # 可选，如["非小细胞肺癌一线", "NSCLC一线"]
biomarker: {生物标志物}      # 可选，如"PD-L1阳性"、"HER2阳性"
created: {YYYY-MM-DD}        # 文件创建日期
updated: {YYYY-MM-DD}        # 最后更新日期
---
```

## 正文结构

### 1. 概述

简要介绍该适应症的治疗背景：

```markdown
## 概述

{适应症名称}的治疗现状：

- **标准治疗**：{当前标准治疗方案}
- **未满足需求**：{主要挑战}
- **热门靶点**：{正在研发的靶点}
```

### 2. 在研药品对比表

**核心表格**：横向对比不同药品的有效性和安全性数据

```markdown
## 在研药品

| 药品 | 公司 | 阶段 | ORR | mPFS | mOS | 安全性要点 | 最新进展 |
|------|------|------|-----|------|-----|-----------|---------|
| [{药品A}](../drug/{药品A}.md) | 公司A | Phase III | 41.4% | 11.3mo | 22.1mo | 3级AE 25% | ASCO 2026 |
| [{药品B}](../drug/{药品B}.md) | 公司B | Phase II | 35.2% | 8.5mo | N/A | 3级AE 18% | ESMO 2025 |

> 来源: [[source/{文件1}.md]] | [[source/{文件2}.md]]
```

**表格列说明**：
- **药品**：链接到药品索引文件
- **公司**：研发公司名称
- **阶段**：临床阶段（Phase I/II/III）
- **ORR/mPFS/mOS**：核心疗效数据，从 source 文件提取
- **安全性要点**：关键不良事件摘要
- **最新进展**：最近数据发布的会议/日期

> **来源说明**: 来源不放在表格列或 frontmatter 中，而是表格下方 `> 来源:` 行。
> 多来源用 `|` 分隔。不在 frontmatter 中使用 sources 数组（Obsidian 兼容性）。

**排序规则**：
- 按临床阶段排序（Phase III → Phase II → Phase I）
- 同阶段按 mPFS 或 ORR 数值排序（高→低）

## 治疗线数处理规则

### 线数分类

| 线数 | 英文缩写 | 文件名示例 | 适用场景 |
|------|---------|-----------|---------|
| 一线 | 1L | `NSCLC_1L.md` | 初治患者 |
| 二线 | 2L | `NSCLC_2L.md` | 一线治疗后进展 |
| 三线及以后 | 3L+ | `NSCLC_3L+.md` | 多线治疗失败 |
| 后线 | Later | `NSCLC_后线.md` | 二线及以后 |

### 线数判断规则

从 source 文件判断治疗线数：
- 检查 `trial_name` 或正文中是否有 "first-line"、"1L"、"一线" 等关键词
- 检查入组人群是否为"初治"、"既往未接受治疗"
- 无法判断时默认为 `1L` 或留空

## 格式检查清单

生成文件后，检查以下项：

- [ ] frontmatter 包含必需字段：indication
- [ ] 文件名与 indication 字段一致
- [ ] 治疗线数（line）正确标注
- [ ] 概述 部分介绍治疗背景
- [ ] 在研药品 表格横向对比有效性和安全性，表格后有 `> 来源:` 行
- [ ] 表格列名一致
- [ ] 药品链接正确（指向 drug/*.md）
- [ ] 来源链接使用 `[[source/...]]` 内联格式
- [ ] 按阶段排序（Phase III优先）
- [ ] 数据时间线 按时间排序
