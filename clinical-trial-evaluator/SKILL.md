# Clinical Trial Evaluator

## ⚠️ 执行约束（触发后必须遵守）

### 强制流程
1. **识别治疗领域**：判断是肿瘤、代谢、自免还是其他领域
2. **提取关键数据**：从用户输入中提取试验设计参数和疗效数据
3. **应用评价框架**：按对应领域的评估维度进行分析
4. **⚠️ 竞争格局分析**：识别SOC/进度最快竞品，进行头对头对比
5. **输出结构化评价**：提供明确的结论、竞争定位和风险提示

### 禁止事项
- ❌ 禁止泛泛而谈，必须针对具体数据点评价
- ❌ 禁止忽略试验设计缺陷
- ❌ 禁止混淆统计显著性与临床意义

## 评价框架（通用 + 领域特化）

### 第一部分：试验设计质量（所有领域通用）

| 维度 | 评估标准 | 权重 |
|------|----------|------|
| **随机化** | 随机 > 非随机（单臂证据等级低） | ⭐⭐⭐⭐⭐ |
| **对照组** | 有对照 > 无对照；活性对照 > 安慰剂 | ⭐⭐⭐⭐⭐ |
| **盲法** | 双盲 > 单盲 > 开放标签 | ⭐⭐⭐⭐ |
| **样本量** | 预设样本量是否充足？是否达到目标入组？ | ⭐⭐⭐⭐ |
| **主要终点** | 是否预先定义？是否为监管认可终点？ | ⭐⭐⭐⭐⭐ |

**设计质量评级：**
- **A级**：随机、双盲、对照、大样本（>300）
- **B级**：随机、开放标签/单盲、有对照
- **C级**：随机、单臂（仅适用于某些罕见病/未满足需求领域）
- **D级**：非随机、单臂、小样本（<50）

### 第二部分：疗效数据评价（按治疗领域）

#### 🔬 肿瘤领域（Oncology）

| 指标 | 评价标准 | 临床意义 |
|------|----------|----------|
| **主要终点达成** | PFS/OS是否显著改善 | 必须达成才能继续评估 |
| **HR（风险比）** | <0.5 优秀；0.5-0.7 良好；0.7-0.85 一般；>0.85 边缘 | 越低越好 |
| **中位PFS延长** | >6个月 优秀；3-6个月 良好；<3个月 有限 | 绝对获益更重要 |
| **中位OS延长** | >6个月 改变实践；3-6个月 有价值；<3个月 有限 | 金标准 |
| **ORR** | >50% 优秀；30-50% 良好；<30% 有限 | 需结合DoR评估 |
| **DoR（缓解持续时间）** | >12个月 持久；6-12个月 中等；<6个月 短暂 | 持久性决定价值 |
| **CR率** | >20% 优秀；10-20% 良好；<10% 一般 | 完全缓解最有价值 |

**肿瘤特有红旗：**
- 🚩 主要终点是ORR而非PFS/OS（监管不接受）
- 🚩 仅亚组阳性而全人群阴性
- 🚩 交叉效应严重（对照组交叉到试验组）
- 🚩 随访时间不足（OS数据不成熟）

---

#### ⚖️ 代谢/减重领域（Metabolic/Obesity）

| 指标 | 评价标准 | 临床意义 |
|------|----------|----------|
| **主要终点达成** | 体重变化%是否显著优于对照 | 必须达成 |
| **体重下降%** | >15% 优秀；10-15% 良好；5-10% 有限；<5% 边缘 | 越高越好 |
| **达标率（如≥5%/≥10%/≥15%）** | >70%达到≥5%；>50%达到≥10% | 反映人群一致性 |
| **绝对体重下降（kg）** | >15kg 优秀；10-15kg 良好 | 结合基线BMI评估 |
| **腰围减少** | >10cm 显著；5-10cm 中等 | 代谢获益指标 |
| **代谢指标改善** | HbA1c、血压、血脂改善幅度 | 综合获益证据 |
| **反弹率** | 停药后体重反弹比例 | 越低越好 |

**减重特有红旗：**
- 🚩 仅短期数据（<24周），无长期维持数据
- 🚩 高停药率（>20%）
- 🚩 严重胃肠道不良反应（>30%）
- 🚩 肌肉量显著下降（>10%）

---

#### 🛡️ 自身免疫领域（Autoimmune）

| 指标 | 评价标准 | 临床意义 |
|------|----------|----------|
| **主要终点达成** | 疾病活动度评分改善是否显著 | 必须达成 |
| **ACR20/50/70**（RA） | ACR50>40% 优秀；ACR20>60% 良好 | 越高越好 |
| **PASI75/90/100**（银屑病） | PASI90>50% 优秀；PASI75>70% 良好 | 皮肤清除度 |
| **临床缓解率** | 达到疾病缓解的患者比例 | 目标治疗策略 |
| **激素减量** | 能否成功减量/停用激素 | 重要获益 |
| **影像学改善**（如mTSS） | 关节结构保护证据 | 长期获益 |
| ** flare-free时间** | 无复发持续时间 | 持久性指标 |

**自免特有红旗：**
- 🚩 短期疗效但高复发率
- 🚩 感染风险显著增加（尤其是严重感染）
- 🚩 免疫原性高（ADA阳性率高）
- 🚩 缺乏头对头对比现有生物制剂

---

#### ❤️ 心血管领域（Cardiovascular）

| 指标 | 评价标准 | 临床意义 |
|------|----------|----------|
| **MACE减少** | HR<0.8 优秀；0.8-0.9 良好 | 硬终点金标准 |
| **心血管死亡降低** | 绝对风险降低>1% | 最有价值终点 |
| **心衰住院减少** | HR<0.75 显著获益 | 重要临床终点 |
| **血压降低** | >10mmHg 显著；5-10mmHg 中等 | 需结合基线 |
| **LDL-C变化** | >50%降低 优秀；30-50% 良好 | 血脂管理 |
| **NT-proBNP改善** | >30%降低 有意义 | 心衰标志物 |

---

#### 🧠 神经/精神领域（CNS）

| 指标 | 评价标准 | 临床意义 |
|------|----------|----------|
| **主要终点改善** | 认知/功能评分变化 | 需为监管认可量表 |
| **效应量（Effect Size）** | >0.5 中等；>0.8 大效应 | Cohen's d |
| **临床改善率** | CDR-SB改善≥1分的患者比例 | AD领域常用 |
| **日常生活能力** | ADCS-ADL等评分改善 | 功能维持重要 |
| **生物标志物** | 脑脊液/影像学生物标志物改善 | 支持疾病修饰 |

### 第三部分：安全性评价（所有领域通用）

| 维度 | 评价标准 | 关注阈值 |
|------|----------|----------|
| **总体TRAE发生率** | 与对照组比较 | 试验组显著更高需警惕 |
| **≥3级AE发生率** | 严重不良事件比例 | >40%需详细评估 |
| **因AE停药率** | 患者因不良反应退出比例 | >10%关注；>15%严重 |
| **致死性AE** | 任何治疗相关死亡 | 必须详细审查 |
| **SAE发生率** | 严重不良事件 | 与对照组比较 |
| **新出现的安全信号** | 非预期的不良反应 | 需特别关注 |

**安全性红旗：**
- 🚩 因AE停药率显著高于对照组（>5%绝对差异）
- 🚩 出现新的严重安全性信号
- 🚩 致死性AE与药物因果关系明确
- 🚩 需要频繁监测/干预才能继续治疗

---

### 第四部分：竞争格局分析（⚠️ 核心评估维度）

**评价原则**：临床数据的价值必须放在竞争格局中评估。与该适应症当前SOC或进度最快的竞品进行头对头比较。

#### 竞争定位判断流程

1. **识别对标药物**
   - 已获批标准治疗（SOC）→ 与SOC比较
   - 无获批药物 → 与临床进度最快的竞品（Ph3/已上市）比较
   - 同类机制 → 与相同靶点/机制的领先药物比较

2. **头对头关键指标对比**

| 对比维度 | 肿瘤 | 减重 | 自免 | 心血管 |
|---------|------|------|------|--------|
| **疗效** | PFS HR、OS HR、ORR | 体重下降%、达标率 | ACR/PASI、缓解率 | MACE HR、CV死亡 |
| **安全性** | ≥3级AE、停药率 | GI AE、停药率 | 感染率、停药率 | SAE、致死性AE |
| **便利性** | 给药频率、给药途径 | 口服vs注射 | 皮下vs静脉 | 口服vs注射 |
| **人群** | 生物标志物选择 | 基线BMI、合并症 | 既往治疗史 | 基线风险 |

3. **差异化价值判断**

| 定位 | 标准 | 商业化前景 |
|------|------|-----------|
| **Best-in-Class** | 疗效显著优于同类（HR差异>15%）或安全性显著更优 | ⭐⭐⭐⭐⭐ |
| **Me-Better** | 疗效相当但安全性/便利性更优 | ⭐⭐⭐⭐ |
| **Me-Too** | 疗效/安全性与现有相当，无明显优势 | ⭐⭐⭐ |
| **Sub-Optimal** | 疗效或安全性劣于现有标准 | ⭐⭐ |

4. **竞争风险信号**
- 🚩 **进度落后**：同类竞品已上市或处于Ph3，本品仅Ph1/2
- 🚩 **疗效相当但更差的安全谱**：难以挑战现有SOC
- 🚩 **仅亚组优势**：全人群数据不支持广泛竞争
- 🚩 **对照组偏弱**：试验设计未选择最强SOC作为对照

---

### 第五部分：临床意义与监管前景

| 评估维度 | 关键问题 |
|----------|----------|
| **未满足需求** | 该领域是否存在重大未满足需求？ |
| **现有治疗对比** | 是否优于/不同于现有标准治疗？ |
| **临床实践改变** | 这个结果能改变临床实践吗？ |
| **监管路径** | 是否达到加速批准/完全批准标准？ |
| **商业化潜力** | 患者人群规模、支付意愿、竞争格局 |

## 使用示例

**用户输入：** "评价这个ASCO数据：某PD-1单抗治疗1L NSCLC（PD-L1≥50%），随机双盲对照试验，N=600，主要终点PFS，HR=0.55，中位PFS 8.5个月 vs 5.2个月，ORR 45% vs 25%，≥3级TRAE 35% vs 28%，因AE停药8% vs 3%"

**评价输出：**
1. 设计质量：A级（随机双盲对照大样本）
2. 疗效：优秀（HR=0.55强获益，PFS延长3.3个月，ORR显著提升）
3. 安全性：可接受（停药率略高但可管理）
4. **竞争格局**：
   - 对标：Keytruda（1L NSCLC SOC，HR=0.55-0.6范围，ORR~45%）
   - 定位：**Me-Too**（疗效相当，需看具体OS数据区分）
   - 风险：PD-1赛道拥挤，差异化空间大
5. 结论：高质量数据但需头对头证明差异化价值

## 注意事项

1. **区分统计显著性与临床意义**：p<0.05不代表临床有价值
2. **关注绝对获益**：相对风险降低（HR）不如绝对获益直观
3. **长期数据优先**：短期替代终点需长期数据验证
4. **人群代表性**：试验人群是否与目标患者人群一致
5. **对照组合理性**：对照组是否为当前最佳标准治疗
6. **⚠️ 竞争格局是核心**：脱离竞争格局谈数据价值是危险的，必须回答"相比现有治疗的优势是什么"
7. **进度就是生命**：同靶点同适应症，落后12个月以上=失去机会
